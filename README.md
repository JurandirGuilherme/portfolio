Meu portfólio pessoal — rápido, acessível e com a minha cara. Construo interfaces
web e mobile com React, Next.js e TypeScript, do protótipo à produção.

## ✨ Destaques

- 🌐 **Bilíngue (PT-BR / EN-US)** — detecta o idioma do navegador e troca na hora, sem recarregar
- 🌗 **Tema claro/escuro** com persistência e sem flash na primeira renderização
- 🎨 **Hero interativo em `<canvas>`** — campo de pontos que reage ao mouse e acompanha o tema
- 📱 **100% responsivo** — do celular ao desktop, com menu que vira hambúrguer no mobile
- ♿ **Acessível** — navegação por teclado, foco visível e respeito a `prefers-reduced-motion`
- ⚡ **SEO** pela Metadata API do Next (Open Graph, Twitter, canonical)

## 🛠️ Stack

`Next.js 16 (App Router)` · `React 19` · `TypeScript` · `CSS Modules` · `next-themes`

Sem framework de CSS nem web fonts externas — fundamentos bem aplicados, com o
conteúdo separado da interface (uma fonte de dados por idioma).

## 📁 Organização

```
app/          layout (SEO/tema), página e estilos globais
components/   layout · sections · ui (canvas, tema, idioma, reveal)
data/         conteúdo por idioma — a fonte única da verdade
lib/i18n/     provider de idioma, detecção e configuração
```

Editar textos, experiências e projetos é só mexer em `data/content/` — sem tocar
nos componentes.
